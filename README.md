# Wx_like 一个网页版微信

Wx_like 采用Angular 2 + ES6 + koa 2 + socketIO 开发 .

### 特征

- 支持网页聊天
- 支持机器人回复


### 快速开发

```bash

npm install
gulp dev

```

浏览器打开     [http://localhost:9000](http://localhost:9000).

### 打包

```bash

gulp pro-dist

```
